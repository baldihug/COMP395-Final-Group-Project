## Goal
Compare the Fourier Neural Operator (FNO) and a U-Net (CNN baseline) on predicting
the time evolution of 2D incompressible Navier-Stokes equations. Evaluation metrics:
relative L2 error and inference speed.

### Dataset Decision
- Evaluated Li et al. (2021) FNO Google Drive dataset — file access unclear/deprecated
- Evaluated PDEBench (doi:10.18419/darus-2986) — full NS dataset is 2.3 TB (274 files
  at ~8 GB each, 512x512 resolution), not practical for a course project
- Selected: neuraloperator Zenodo dataset (doi:10.5281/zenodo.12825163)
  - File: nsforcing_128.tgz (~1.5 GB)
  - Resolution: 128x128
  - Source: https://zenodo.org/records/12825163
  - Format: PyTorch .pt tensors (vorticity fields)
  - Rationale: manageable size, standard benchmark resolution, directly citable,
    matches resolution used in Li et al. FNO paper experiments

### Dataset Download
- Downloaded nsforcing_128.tgz to data/ directory
- Extracted contents:
  - data/nsforcing_train_128.pt — training set (1.3 GB)
  - data/nsforcing_test_128.pt  — test set (251 MB)
- Data files excluded from git via .gitignore (too large for GitHub)
